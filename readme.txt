Installation: 
  edit path.*.txt files to point to installation folder for SPM/CONN/SPM_SS packages
  edit ver.*.txt files to specify valid version/release numbers for SPM/CONN/SPM_SS packages

Usage:
  evlab17 init            : initializes evlab17 functions (run this at the beginning of each new Matlab session)
  evlab17 run_preproc ... : preprocessing functions (see HELP EVLAB_RUN_PREPROC)
  evlab17 run_model ...   : first-level analysis functions (see HELP EVLAB_RUN_MODEL)
  evlab17 spm_ss ...      : subject-specific toolbox functions (see HELP SPM_SS)
